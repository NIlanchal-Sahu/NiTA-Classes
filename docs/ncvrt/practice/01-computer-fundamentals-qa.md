## Section B — Short Answer (20)

| # | Question | Model Answer |
|---|----------|--------------|
| 1 | Define computer. | Electronic device: input → process → store → output using programs. |
| 2 | Four hardware components? | CPU, RAM, keyboard, monitor (any four valid). |
| 3 | RAM vs ROM? | RAM volatile temporary; ROM non-volatile permanent boot instructions. |
| 4 | Input vs output? | Input sends data in (keyboard); output shows results (monitor). |
| 5 | Convert decimal 9 to binary? | 1001 |
| 6 | Convert binary 1010 to decimal? | 10 |
| 7 | What is file extension? | Suffix after dot identifying file type (.docx). |
| 8 | Types of computers? | Micro, mini, mainframe, supercomputer. |
| 9 | What is CPU speed measured in? | GHz (gigahertz). |
| 10 | Name storage devices. | HDD, SSD, pen drive, DVD, memory card. |
| 11 | Data vs information? | Data raw; information processed meaningful data. |
| 12 | What is booting? | Starting/loading OS when computer turns on. |
| 13 | Full form CPU, ALU, CU? | Central Processing Unit; Arithmetic Logic Unit; Control Unit. |
| 14 | 1 MB = ? KB | 1024 KB |
| 15 | Mouse functions? | Click select, double-click open, right-click menu, scroll. |
| 16 | System vs application software? | System runs hardware (OS); application does user tasks (Word). |
| 17 | What is multitasking? | Running multiple programs simultaneously. |
| 18 | Safe shutdown steps? | Save work → Start → Shut down → wait → power off. |
| 19 | What is virus? | Malicious program that can spread and damage data/system. |
| 20 | Why backup? | Recover data if disk fails, virus, or accidental delete. |
