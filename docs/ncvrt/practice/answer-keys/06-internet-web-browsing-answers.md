# Answer Key — Module 06: Internet & Web Browsing

## MCQ Answer Key

1. Internet is?
   a) Global network of computers
   b) One PC
   c) Printer
   d) Keyboard
   **Answer: a) Global network of computers**
   **Explanation:** The Internet is a global network connecting millions of computers worldwide to share data and services.

2. WWW stands for?
   a) World Wide Web
   b) World Web Wide
   c) Wide World Web
   d) Web World Wide
   **Answer: a) World Wide Web**
   **Explanation:** WWW (World Wide Web) is the collection of web pages and links accessed through browsers on the Internet.

3. Browser example?
   a) Google Chrome
   b) MS Word
   c) Excel
   d) Access
   **Answer: a) Google Chrome**
   **Explanation:** Google Chrome — this is how browsers and Internet connectivity work in everyday use.

4. URL stands for?
   a) Uniform Resource Locator
   b) Universal Reference Link
   c) Unified Resource Line
   d) User Resource Locator
   **Answer: a) Uniform Resource Locator**
   **Explanation:** A URL (Uniform Resource Locator) is the web address you type in the browser — e.g. https://www.google.com.

5. ISP provides?
   a) Internet connection
   b) Word license
   c) Printer
   d) CPU
   **Answer: a) Internet connection**
   **Explanation:** Internet connection is the correct answer. Word license and Printer are not the right choices here. This is an important CCC exam point — memorise it.

6. HTTPS indicates?
   a) Secure connection
   b) Slow site
   c) No images
   d) Free Wi-Fi
   **Answer: a) Secure connection**
   **Explanation:** HTTPS encrypts data between your browser and the website — look for the padlock when entering passwords or paying online.

7. HTTP stands for?
   a) Hyper Text Transfer Protocol
   b) High Transfer Text Program
   c) Hyper Tool Transfer
   d) Home Text Protocol
   **Answer: a) Hyper Text Transfer Protocol**
   **Explanation:** HTTP/HTTPS are protocols browsers use to load web pages — HTTPS adds encryption for security.

8. Download means?
   a) Save from internet to PC
