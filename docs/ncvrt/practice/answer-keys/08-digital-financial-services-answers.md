# Answer Key — Module 08: Digital Financial Services

## MCQ Answer Key

1. UPI stands for?
   a) Unified Payments Interface
   b) Universal Payment Internet
   c) Unified Personal ID
   d) User Payment Index
   **Answer: a) Unified Payments Interface**
   **Explanation:** UPI (Unified Payments Interface) lets you transfer money instantly between bank accounts using a mobile app and UPI PIN.

2. BHIM app developed by?
   a) NPCI / Government initiative
   b) Microsoft
   c) Google only
   d) Apple only
   **Answer: a) NPCI / Government initiative**
   **Explanation:** BHIM (Bharat Interface for Money) is a UPI app launched with support from NPCI — the organisation that runs UPI payment infrastructure in India.

3. Net banking allows?
   a) Banking via internet
   b) Only ATM cash
   c) Only postal mail
   d) Only phone call branch only always
   **Answer: a) Banking via internet**
   **Explanation:** Net banking (online banking) lets you check balance, transfer money, and pay bills through your bank's official website or app — without visiting a branch.

4. Digital wallet example?
   a) Paytm, PhonePe
   b) Notepad
   c) Paint
   d) Calculator
   **Answer: a) Paytm, PhonePe**
   **Explanation:** Paytm and PhonePe are digital wallets/UPI apps used for cashless payments, recharges, and bill pay from your phone.

5. AEPS uses?
   a) Aadhaar authentication for payments
   b) Email password only
   c) Word doc
   d) Excel chart

   **Explanation:** AEPS (Aadhaar Enabled Payment System) uses Aadhaar-based authentication so people can do banking with fingerprint verification.

   d) Only Total Price

   **Explanation:** OTP (One Time Password) is a temporary code sent to your phone for secure login or payment — never share it with anyone.
