# Answer Key — Module 10: Cyber Security

## MCQ Answer Key

1. Virus is?
   a) Malicious software
   b) Hardware part
   c) Input device
   d) Browser feature
   **Answer: a) Malicious software**
   **Explanation:** A computer virus is harmful software that can copy itself and damage files or the system. It is not hardware.

2. Malware means?
   a) Malicious software general
   b) Helpful tool
   c) OS update
   d) Printer driver always safe wrong
   **Answer: a) Malicious software general**
   **Explanation:** Malware (including viruses) is malicious software designed to harm, steal data, or take control of your device.

3. Phishing is?
   a) Fraudulent data theft attempt
   b) Fishing hobby only
   c) Antivirus brand
   d) Browser cache
   **Answer: a) Fraudulent data theft attempt**
   **Explanation:** Phishing is a fraud trick using fake emails, links, or messages to steal passwords, OTP, or bank details.
