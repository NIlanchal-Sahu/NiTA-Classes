## Section B — Model Answers (key points)

**21.** Block diagram: Input devices → CPU → Memory ↔ Storage → Output devices. Label all five blocks.

**22.** Hardware: physical (CPU, keyboard, monitor). Software: programs (Windows, Word, Chrome). Hardware needs software to work.

**23.** RAM: temporary, volatile, holds running programs. ROM: permanent instructions for boot, non-volatile.

**24.** Any eight: Ctrl+C Copy, Ctrl+V Paste, Ctrl+X Cut, Ctrl+Z Undo, Ctrl+S Save, Win+E Explorer, Win+L Lock, Alt+Tab Switch, F2 Rename, Delete to Recycle Bin.

**25.** File Explorer → navigate → right-click New Folder → create file in Notepad → Save As → Ctrl+C file → open destination → Ctrl+V → F2 rename.

## Section C — Model Answers (key points)

**26.** CPU: ALU (calculations), CU (control), registers. Input: keyboard, mouse, scanner. Output: monitor, printer, speaker. Storage: HDD, SSD, pen drive. Backup protects against data loss from failure or virus.

**27.** OS functions: process, memory, file, device management, user interface. File ops: create folder, copy (Ctrl+C/V), move (Ctrl+X/V), delete (Recycle Bin), restore from Recycle Bin. Security: Win+L lock, strong password, log out on shared PC, no unknown USB.
