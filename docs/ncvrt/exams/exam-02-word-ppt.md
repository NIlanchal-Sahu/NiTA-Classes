# Exam 2 — MS Word & MS PowerPoint

**Unit:** 2 (30 hours) | **Time:** 2 hours theory + 1.5 hours practical  
**Total Marks:** 100 (Theory 50 + Practical 50)  
**Pass Marks:** 50

---

## Section A — MCQ (1 × 15 = 15)

1. Default Word file extension — a) .docx  b) .pptx  c) .xlsx  d) .pdf  
2. Shortcut for Save — a) Ctrl+P  b) Ctrl+S  c) Ctrl+O  d) Ctrl+N  
3. Page break shortcut — a) Ctrl+Enter  b) Alt+Enter  c) Shift+Enter  d) F5  
4. Mail merge tab — a) Home  b) Insert  c) Mailings  d) Review  
5. Justify alignment — a) Both edges aligned  b) Center only  c) Left only  d) Right only  
6. PowerPoint file extension — a) .docx  b) .pptx  c) .accdb  d) .txt  
7. Start slideshow from beginning — a) F5  b) F7  c) Ctrl+F5  d) Alt+F5  
8. Transition applies to — a) Objects on slide  b) Between slides  c) Notes  d) Handouts  
9. Animation applies to — a) Between slides  b) Slide background  c) Objects on slide  d) Printer  
10. Insert page number — a) Home  b) Insert  c) Design  d) View  
11. Find and Replace — a) Ctrl+F / Ctrl+H  b) Ctrl+S  c) Ctrl+B  d) F2  
12. 6×6 rule relates to — a) Excel rows  b) Readable slides  c) Print margins  d) Mail merge  
13. Export Word to PDF — a) Save As → PDF  b) Print only  c) Insert PDF  d) Mailings  
14. Duplicate slide — a) Right-click → Duplicate  b) Ctrl+D always  c) Delete slide  d) F5  
15. Wrap text for image — a) Picture Format  b) Home font  c) Transitions  d) Slide Show
