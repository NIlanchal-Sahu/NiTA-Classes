# Exam 1 — Computer Fundamentals & Operating System

**Unit:** 1 (30 hours) | **Time:** 2 hours theory + 1 hour practical  
**Total Marks:** 100 (Theory 60 + Practical 40)  
**Pass Marks:** 50

---

## Section A — Multiple Choice Questions (1 mark each × 20 = 20)

1. CPU stands for —  
   a) Central Process Unit  b) Central Processing Unit  c) Computer Personal Unit  d) Control Processing Unit

2. Which is volatile memory?  
   a) ROM  b) Hard disk  c) RAM  d) DVD

3. Which is an input device?  
   a) Monitor  b) Printer  c) Scanner  d) Speaker

4. 1 GB equals approximately —  
   a) 1000 KB  b) 1024 MB  c) 100 MB  d) 1024 KB

5. Software is —  
   a) Physical component  b) Set of programs  c) Monitor  d) Keyboard

6. Binary number system uses digits —  
   a) 0–9  b) 0–1  c) 0–7  d) 0–F

7. Which stores permanent boot instructions?  
   a) RAM  b) Cache  c) ROM  d) Register

8. GUI stands for —  
   a) General User Internet  b) Graphical User Interface  c) Graphic Utility Input  d) Group User ID

9. Shortcut to open File Explorer —  
   a) Win + E  b) Win + R  c) Ctrl + E  d) Alt + E

10. Delete key sends file to —  
    a) Permanent delete  b) Recycle Bin  c) Desktop  d) Control Panel

11. Which is system software?  
    a) MS Word  b) Windows 10  c) Chrome  d) Tally

12. ALU is part of —  
    a) Monitor  b) CPU  c) Printer  d) Keyboard

13. Pen drive is —  
    a) Input device  b) Output device  c) Storage device  d) Processing device

14. Warm boot means —  
    a) First start of day  b) Restart without full power off  c) Shutdown  d) Sleep mode

15. Win + L is used to —  
    a) Log off  b) Lock computer  c) Open library  d) Launch Word

16. Invalid character in Windows file name —  
    a) _  b) -  c) *  d) .

17. Task Manager opens with —  
    a) Ctrl+Alt+Del menu  b) Ctrl+Shift+Esc  c) Both a and b  d) Alt+F4

18. Copy shortcut is —  
    a) Ctrl+X  b) Ctrl+C  c) Ctrl+V  d) Ctrl+Z

19. File extension for Word 2016+ —  
    a) .txt  b) .doc  c) .docx  d) .xlsx

20. Data processed into meaningful form is called —  
    a) Input  b) Output  c) Information  d) Hardware
