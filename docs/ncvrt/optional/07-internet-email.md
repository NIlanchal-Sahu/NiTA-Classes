# Subject 7: Internet & Email (12 Hours)

## Learning Objectives

- Explain internet, WWW, browser, URL, ISP
- Connect to internet (Wi-Fi, mobile data)
- Browse websites safely and use search engines
- Create and use email (compose, attach, reply, forward)
- Understand cyber safety, passwords, phishing
- Use common online services (forms, downloads, cloud storage)

---

## 1. Internet Basics

**Internet** = global network of connected computers  
**WWW (World Wide Web)** = pages linked by hyperlinks (HTTP/HTTPS)  
**Website** = collection of web pages on a server  
**Web page** = single HTML document  
**Home page** = first page of a website

### Key terms
| Term | Meaning |
|------|---------|
| Browser | Software to view web — Chrome, Edge, Firefox |
| URL | Web address — `https://www.example.com` |
| ISP | Internet Service Provider — gives connection |
| Modem/Router | Device connecting home to ISP |
| Wi-Fi | Wireless network |
| Download | Save file from internet to PC |
| Upload | Send file from PC to internet |
| Cloud | Servers on internet — Google Drive, OneDrive |

---

## 2. Web Browser

### Popular browsers
Google Chrome, Microsoft Edge, Mozilla Firefox, Safari

### Parts of browser window
- **Address bar** — type URL or search
- **Tabs** — multiple pages open (Ctrl+T new tab)
- **Bookmarks / Favorites** — save sites (Ctrl+D)
- **History** — past visited pages (Ctrl+H)
- **Downloads** — Ctrl+J

### Navigation
- Back / Forward / Refresh (F5)
- **HTTPS** — secure connection (padlock icon) — use for banking, login

---

## 3. Search Engines

**Google, Bing, DuckDuckGo** — find information by keywords

### Effective searching
- Use specific keywords: `"NCVRT syllabus DCA"` not just `computer`
- **Quotes** — exact phrase
- **site:** — `site:gov.in digital locker`
- Evaluate sources — prefer .gov.in, .edu, official sites

---

## 4. Email (Electronic Mail)

**Email address format:** `username@domain.com`  
Example: `student@gmail.com`

### Free email providers
Gmail (Google), Outlook (Microsoft), Yahoo Mail

### Email clients vs webmail
- **Webmail:** Browser — mail.google.com
- **Client:** Outlook app, Thunderbird — downloads mail to PC

---

## 5. Email Operations

| Action | Steps |
|--------|-------|
| Compose | New message → To, Subject, Body → Send |
| Reply | Answer sender only |
| Reply All | Answer all recipients |
| Forward | Send copy to someone else |
| Attachment | Paperclip → attach file (max size limit ~25 MB Gmail) |
| CC | Carbon copy — visible to all |
| BCC | Blind copy — hidden from others |

### Email etiquette
- Clear subject line
- Formal greeting in official mail
- Do not write password in email
- Avoid ALL CAPS (looks like shouting)

## 7. Cyber Safety

### Threats
| Threat | Description | Protection |
|--------|-------------|------------|
| Virus/Malware | Harmful software | Antivirus, don't open suspicious attachments |
| Phishing | Fake emails/sites steal password | Check URL, never click unknown links |
| Spam | Unwanted bulk email | Mark as spam, don't reply |
| Identity theft | Stealing personal data | Strong password, 2FA |

### Strong password rules
- 8+ characters, mix upper/lower/number/symbol
- Different password per site
- Enable **Two-Factor Authentication (2FA)** on Gmail

### Safe browsing habits
- Log out on shared computers
- Do not download pirated software
- Verify HTTPS on payment pages
- Keep Windows and browser updated
