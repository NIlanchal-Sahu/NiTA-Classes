## Section B — Short (20) — key points

1. Create presentation: Blank → add slides → save .pptx  
2. Apply theme: Design → Themes  
3. Add transition: Transitions tab → choose → Apply To All  
4. Animate bullets one-by-one: Animation → By Paragraph / Animation Pane On Click  
5. Insert picture: Insert → Pictures  
6. Run slideshow: F5  
7. Print handouts 6/page: File Print → Handouts 6 slides  
8. Export PDF: Save As PDF  
9. Duplicate vs new slide difference  
10. Title and Content layout use  
11. SmartArt vs Shapes  
12. Presenter View enable path  
13. Slide size change when needed (projector)  
14. Video embed cautions (file size)  
15. Consistent fonts rule  
16. Transition duration meaning  
17. Motion path animation  
18. .ppsx vs .pptx  
19. Reorder in Slide Sorter  
20. Design Ideas purpose
