## Section B — Short Answer (20) — summaries

1. Open/save doc: Start Word → type → Ctrl+S → choose path.  
2. Format paragraph: alignment, spacing, indent, bullets — Home tab.  
3. Insert table 3×4: Insert → Table → grid.  
4. Mail merge steps: Mailings → Start → Recipients → Fields → Finish.  
5. Header/footer: Insert → Header/Footer → edit → close.  
6. Find/Replace "old"→"new": Ctrl+H.  
7. Export PDF: File → Save As → PDF.  
8. Page orientation change: Layout → Orientation.  
9. Picture wrap Square: select image → Picture Format → Wrap Text.  
10. Styles benefit: consistent headings/formatting.  
11. Line break vs paragraph: Shift+Enter vs Enter.  
12. Print preview: Ctrl+P preview pane.  
13. Merge fields examples: Name, Address, City.  
14. Justify vs left align difference.  
15. Table Tab key behavior.  
16. Spell check red underline meaning.  
17. Save As vs Save difference.  
18. Page break vs section break.  
19. Columns for newsletter: Layout → Columns.  
20. Professional letter parts: date, address, subject, body, signature.
