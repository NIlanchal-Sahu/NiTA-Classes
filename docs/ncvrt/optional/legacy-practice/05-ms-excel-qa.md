## Section B — Short (20)

| # | Question | Answer |
|---|----------|--------|
| 1 | Formula total B2:D2? | `=SUM(B2:D2)` |
| 2 | Average marks B2:B11? | `=AVERAGE(B2:B11)` |
| 3 | Pass if >=40 in C2? | `=IF(C2>=40,"Pass","Fail")` |
| 4 | % marks 245/500 in D2? | `=245/500*100` or `=B2/500*100` |
| 5 | Count numeric A2:A20? | `=COUNT(A2:A20)` |
| 6 | Relative vs absolute? | Relative adjusts; $ fixed |
| 7 | Create column chart steps? | Select data → Insert → Column |
| 8 | Highlight <40 red? | Conditional Formatting → Less Than 40 |
| 9 | Sort by column C? | Select data → Data Sort |
| 10 | Filter enable? | Data → Filter |
| 11 | Print repeat row 1? | Page Layout → Print Titles → Rows $1:$1 |
| 12 | Merge & center title? | Select → Merge & Center |
| 13 | COUNT vs COUNTA? | COUNT numbers; COUNTA non-blank |
| 14 | SUMIF example? | `=SUMIF(A:A,"Delhi",B:B)` |
| 15 | Nested IF 60/45? | `=IF(B2>=60,"First",IF(B2>=45,"Second","Fail"))` |
| 16 | TODAY use? | Current date auto-updates |
| 17 | ROUND 3.456 to 2 dp? | `=ROUND(3.456,2)` → 3.46 |
| 18 | Link Sheet2 A1? | `=Sheet2!A1` |
| 19 | Print area set? | Select → Page Layout → Print Area → Set |
| 20 | #DIV/0! fix? | Check divisor cell not zero |
