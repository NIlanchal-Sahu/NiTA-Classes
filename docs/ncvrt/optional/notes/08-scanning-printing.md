# Subject 8: Scanning & Printing Documents (8 Hours)

## Learning Objectives

- Connect and configure printer and scanner
- Print documents from Word, Excel, PowerPoint, PDF
- Use print settings (copies, duplex, page range, orientation)
- Scan documents to image and PDF
- Use OCR concept (scan to editable text)
- Troubleshoot common printer/scanner problems

---

## 1. Printers — Types and Use

| Type | Technology | Best for | Cost |
|------|------------|----------|------|
| Inkjet | Liquid ink spray | Photos, colour, home | Medium ink cost |
| Laser | Toner powder | Office text, high volume | Lower per page |
| Dot matrix | Impact pins | Bills, multi-copy forms | Old technology |
| Thermal | Heat paper | Receipts | Shops |

**Multifunction Device (MFD)** = Print + Scan + Copy in one machine

---

## 2. Connecting a Printer

### USB printer
1. Connect USB cable to PC and printer
2. Power on printer
3. Windows auto-installs driver (or use CD/manufacturer website)
4. Settings → Bluetooth & devices → Printers & scanners → Add device

### Network / Wi-Fi printer
- Connect printer to same Wi-Fi as PC
- Add printer by IP address or discovery in Settings

**Set as default printer:** Settings → Printers → Manage → Set as default

---

## 3. Printing Documents

### General steps (Word/Excel/PPT/PDF)
1. **Ctrl + P** or File → Print
2. Select **printer**
3. Choose settings:

| Setting | Options |
|---------|---------|
| Copies | Number of sets |
| Pages | All, current, custom range (1,3,5-8) |
| Orientation | Portrait / Landscape |
| Paper size | A4, Letter |
| Duplex | Print both sides (if supported) |
| Collated | 1,2,3 then 1,2,3 vs 1,1,2,2,3,3 |
| Color | Color / Grayscale |

4. **Print preview** — check layout before printing
5. Click **Print**

### Excel-specific
- Print active sheet / entire workbook / selection
- Page Layout → Print Area
- Scale to fit one page wide

### PowerPoint-specific
- Print full slides, handouts (2/6 slides per page), notes pages

---

## 4. Scanning Documents

### Using scanner glass (flatbed)
1. Place document face-down on glass, align to corner marker
2. Open **Windows Scan** app or manufacturer app (HP Scan, Epson Scan)
3. Choose:
   - **File type:** JPEG, PNG, PDF
   - **Resolution (DPI):** 150 web, 300 standard print, 600 high quality
   - **Color mode:** Color / Grayscale / Black & white
4. Preview → Scan → Save to folder

### Using ADF (Automatic Document Feeder)
- Stack multiple pages in feeder tray  
- Scan all pages to single PDF

### Scan to email / folder
Many MFD apps allow direct scan-to-email or scan-to-USB

---

## 5. OCR — Scan to Editable Text

**OCR (Optical Character Recognition)** converts image of text → editable text

- **OneNote:** Insert printout → right-click → Copy Text from Picture
- **Word:** Insert → Pictures → scanned image → File → Save As → searchable PDF (Office 365)
- **Online tools / Adobe Scan app** — mobile OCR

Use 300 DPI, straight alignment, good lighting for best OCR

---

## 6. Photocopy (MFD)

1. Place original on glass or ADF
2. Select copy on printer panel
3. Choose copies, color/B&W, size (A4)
4. Start — collect copies from output tray

---

## 7. File Formats for Scan/Print

| Format | Use |
|--------|-----|
| PDF | Best for documents, email, archive |
| JPEG | Photos, smaller size |
| PNG | Screenshots, lossless |
| TIFF | High quality archive |

**Naming scans:** `2026-06-09_admission-form_student-name.pdf`

---

## 8. Troubleshooting

| Problem | Solution |
|---------|----------|
| Printer offline | Check power, cable, Wi-Fi; set online in Settings |
| Paper jam | Follow printer panel instructions; remove torn pieces |
| Blank pages | Check toner/ink levels; remove protective tape on new cartridge |
| Wrong layout | Check orientation, margins, print preview |
| Cannot scan | Reinstall scanner driver; use Windows Scan app |
| Queue stuck | Settings → Printers → Open queue → Cancel all → retry |
| Poor scan quality | Clean glass; increase DPI; flatten document |
