# Spreadsheet — Practical Lab Pack (CCC)

**Software:** MS Excel or LibreOffice Calc  
**Total labs:** 6 · **Time:** ~12 hours practical

---

## Lab 1: Data Entry (2 hrs)

1. Create workbook `CCC_Marks.xlsx`  
2. Columns: Name, Roll, Sub1, Sub2, Sub3  
3. Enter 8 students with sample marks (0–100)  
4. Format header row bold, centre

---

## Lab 2: Formulas (2 hrs)

1. Column **Total** = sum of 3 subjects  
2. Column **Average** = average of 3 subjects  
3. Column **Result** = IF average >= 40, "Pass", else "Fail"  
4. Verify by changing one mark — formulas update

---

## Lab 3: Functions (2 hrs)

1. Below table calculate:  
   - Highest marks (MAX)  
   - Lowest marks (MIN)  
   - Class average (AVERAGE)  
   - Count pass students (COUNTIF)  
2. Label each result cell

---

## Lab 4: Sort & Filter (1.5 hrs)

1. Sort table by Total descending  
2. Enable filter — show only Pass students  
3. Filter one subject marks > 75

---

## Lab 5: Conditional Formatting (1.5 hrs)

1. Highlight marks < 40 in red  
2. Highlight marks >= 75 in green  
3. Data bars on Total column (optional)

---

## Lab 6: Chart (3 hrs)

1. Column chart — student names vs Total marks  
2. Pie chart — Pass vs Fail count  
3. Add chart titles and axis labels  
4. Set print area; repeat header row; print preview
