# Presentation — Practical Lab Pack (CCC)

**Software:** MS PowerPoint or LibreOffice Impress  
**Total labs:** 4 · **Time:** ~8 hours practical

---

## Lab 1: 5-Slide "About Yourself" (3 hrs)

Create presentation with **exactly 5 slides:**

| Slide | Content |
|-------|---------|
| 1 | Title: Your Name — CCC Student (Title Slide layout) |
| 2 | About Yourself — photo + 3 bullet points |
| 3 | Education — school/college, course |
| 4 | Skills — computer skills list |
| 5 | Career Goal + Thank You |

Apply one **theme** to all slides.

---

## Lab 2: Transitions & Animation (2 hrs)

1. Same presentation — add **same transition** to all slides  
2. Animate bullet points on slide 2 — appear one by one  
3. Run slideshow F5; practice Esc to exit

---

## Lab 3: SmartArt / Image (1.5 hrs)

1. New slide with **SmartArt** process diagram (3 steps: Learn → Practice → Pass CCC)  
2. Insert image from device with caption  
3. Save as PDF

---

## Lab 4: Handouts & Export (1.5 hrs)

1. Print preview — **6 slides per page** handout  
2. Save as `.ppsx` (or Impress slideshow export)  
3. Export video/mp4 if software supports
