# Word Processing — Practical Lab Pack (CCC)

**Software:** MS Word 2016+ or LibreOffice Writer  
**Total labs:** 6 · **Time:** ~12 hours practical

---

## Lab 1: Basic Document (2 hrs)

1. Open blank document  
2. Title **"My Introduction"** — Heading 1, centre align  
3. Three paragraphs about yourself — justify, 1.5 line spacing  
4. Save as `Lab1_Introduction.docx` (or `.odt`)

**Check:** Font size 12 body, title larger, file opens correctly

---

## Lab 2: Formatting (2 hrs)

1. Open Lab 1 file  
2. Bold your name; italic your school/institute; underline email  
3. Apply bullet list of 5 hobbies  
4. Numbered list of 3 goals for CCC course  
5. Save As PDF

---

## Lab 3: Table (2 hrs)

1. Create **4×4 table** — Student Details  
2. Rows: Name, Address, Phone, Email (Field | Detail)  
3. Shade header row; borders all cells  
4. Save as `Lab3_StudentTable.docx`

---

## Lab 4: Formal Letter (2 hrs)

1. Create one-page **application letter** to Principal/Director  
2. Subject: Admission to DCA/CCC course  
3. Date top right; proper greeting; signature line  
4. Header: institute name; footer: page number  
5. Save + print preview

---

## Lab 5: Resume (2 hrs)

1. Use template or create from scratch  
2. Sections: Objective, Education, Skills, Contact  
3. Insert small photo (optional) with wrap text  
4. Export PDF for submission practice

---

## Lab 6: Mail Merge (2 hrs)

1. Create Excel list: 5 names + cities  
2. Word/Writer mail merge letter: "Dear «Name», Greetings from «City»..."  
3. Preview and generate 5 letters  
4. Print or save merged PDF
