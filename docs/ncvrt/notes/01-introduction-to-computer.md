## 2. Topic Explanation

### 2.1 Introduction to Computers

A **computer** is an electronic device that accepts **input**, processes it according to instructions called **programs**, stores data when required, and produces meaningful **output**. In simple words, a computer helps us convert raw facts into useful results.

For example, when a student enters marks of five subjects into a spreadsheet, the marks are input. The spreadsheet formula calculates total and average, which is processing. The saved file is storage. The displayed result or printed report is output.

Computers are important because they perform tasks with speed, accuracy, storage capacity, and consistency. A human may take several minutes to calculate hundreds of marks, but a computer can do it within seconds. Computers also help people communicate, learn, manage accounts, create documents, access government services, and control machines.

#### Basic Working Cycle of a Computer

```text
Input -> Processing -> Storage -> Output
              |
           Memory
```

- **Input:** Data or instructions given to a computer. Examples: typing text, clicking a mouse, scanning a document, touching a mobile screen.
- **Processing:** The CPU works on the input according to software instructions. Examples: calculating marks, opening a file, converting typed letters into a document.
- **Memory:** Temporary working area used while processing. RAM is the main temporary memory.
- **Storage:** Long-term saving of files and programs. Examples: HDD, SSD, pen drive, memory card.
- **Output:** Result produced by the computer. Examples: text on monitor, printed certificate, sound from speaker, downloaded PDF.

#### Why This Concept Is Important

The input-process-output-storage model is the foundation for understanding all computer systems. Whether you use a desktop, laptop, smartphone, ATM, railway ticket machine, or hospital billing system, the same basic cycle is followed.

#### Types of Computers

| Type | Description | Main Use | Examples |
|---|---|---|---|
| Microcomputer | Small computer for one user | Home, office, school, shop | Desktop, laptop, tablet |
| Workstation | Powerful single-user computer | Design, engineering, media editing | CAD workstation |
| Server | Computer that provides services to many users | Websites, LMS, banking, file sharing | Web server, database server |
| Mainframe | Very large and powerful computer | Large organizations and high-volume processing | Banks, railways, insurance |
| Supercomputer | Extremely high-speed computer | Weather, research, AI, simulations | PARAM, Fugaku, Frontier |
| Embedded computer | Computer built into a device | Control and automation | ATM, washing machine, smart TV |

#### Historical Perspective

Early calculating devices such as the abacus and mechanical calculators performed limited arithmetic. Modern electronic computers developed when electricity, electronic circuits, and stored programs were introduced. Today, computers have become smaller, faster, cheaper, and connected to the internet. Future computers will use more artificial intelligence, cloud computing, quantum computing research, and automation.

#### Common Misconceptions

- The **CPU** is not the full computer cabinet. The CPU is a processor chip inside the system unit.
- A computer is not useful only for typing. It is used for communication, data analysis, design, banking, learning, entertainment, and automation.
- More storage does not always mean faster performance. Performance depends on CPU, RAM, storage speed, software, and operating system.

#### Did You Know?

Modern smartphones often have more processing power and storage than many early room-sized computers.

### 2.3 Evolution of Computers: Five Generations

Computer generations explain how computer technology changed over time. Each generation is mainly identified by the electronic technology used inside computers.

| Generation | Period | Main Technology | Features | Examples |
|---|---|---|---|---|
| First Generation | 1940s-1950s | Vacuum tubes | Very large, high heat, high electricity, slow | ENIAC, UNIVAC |
| Second Generation | 1950s-1960s | Transistors | Smaller, faster, more reliable | IBM 1401 |
| Third Generation | 1960s-1970s | Integrated Circuits (ICs) | Multiprogramming, cheaper, more compact | IBM System/360 |
| Fourth Generation | 1970s-present | Microprocessors, VLSI | Personal computers, GUI, networking | Intel PCs, laptops |
| Fifth Generation | Present and future | AI, ULSI, parallel processing | Smart systems, voice assistants, robotics | AI systems, supercomputers |

#### First Generation: Vacuum Tubes

Vacuum tubes acted like electronic switches. These computers were very large, produced heat, consumed much electricity, and were difficult to maintain. They were mainly used by governments, military, and research institutions.

#### Second Generation: Transistors

Transistors replaced vacuum tubes because they were smaller, faster, cheaper, and more reliable. This made computers more practical for business and scientific work.

#### Third Generation: Integrated Circuits

An **Integrated Circuit (IC)** places many electronic components on a small chip. This reduced size and cost. Computers became more reliable and could run multiple programs more effectively.

#### Fourth Generation: Microprocessors

A **microprocessor** places the CPU on a single chip. This led to personal computers, laptops, graphical user interfaces, and widespread computer use in schools, offices, homes, banks, and shops.

#### Fifth Generation: Artificial Intelligence

Fifth-generation computing focuses on artificial intelligence, natural language processing, machine learning, robotics, voice recognition, and intelligent decision-making. Examples include chatbots, recommendation systems, smart assistants, fraud detection systems, and autonomous vehicles.

#### Memory Trick

**Vacuum Tube -> Transistor -> IC -> Microprocessor -> AI**

Mnemonic: **Very Talented Indians Make Advances**

#### Exam Tip

Fifth-generation computers are related to **AI and intelligent systems**. This is different from **5G mobile network**, which is a communication technology.

#### Common Mistakes

- Do not confuse 5G mobile with fifth-generation computers.
- Do not write "internet" as the main technology of fourth-generation computers. The key technology is the microprocessor.
- Do not say vacuum tubes are modern. They belong to the first generation.

#### Did You Know?

ENIAC occupied a large room and consumed huge electricity, while today a small smartphone can perform many tasks faster.

### 2.4 IT Gadgets and Mobile Computing

An **IT gadget** is an electronic device used for computing, communication, storage, learning, or digital services. Mobile computing means using portable devices to access information and services from different locations.

#### Smartphones

A **smartphone** is a pocket-sized computer with a touchscreen, processor, memory, camera, speaker, microphone, GPS, sensors, internet access, and mobile operating system such as Android or iOS.

Smartphones are used for calling, messaging, online classes, UPI payments, railway booking, DigiLocker, telemedicine, learning apps, maps, photography, and email. Mobile apps are application software installed from app stores.

#### Tablets

A **tablet** is a portable touchscreen device larger than a smartphone and lighter than most laptops. Tablets are useful for reading e-books, attending video classes, writing notes with a stylus, watching tutorials, and showing presentations.

#### IoT Devices

**IoT** means **Internet of Things**. It refers to everyday objects connected to the internet so they can collect data, send information, or be controlled remotely.

Examples include smart bulbs, CCTV cameras, fitness bands, smart TVs, home assistants, smart meters, and GPS trackers. IoT devices usually need internet, sensors, and mobile apps.

#### Other Common Gadgets

#### Smartphone vs Tablet

| Point | Smartphone | Tablet |

### 2.5 Components of a Computer System

A computer system is made of hardware, software, data, users, and procedures. Hardware alone cannot work without software, and software needs hardware to run.

#### Visual Learning: Information Flow

```text
Keyboard/Mouse/Scanner
        |
        v
Input -> CPU Processing -> Output
        |        |             |
        |        v             v
        |      Memory       Monitor/Printer/Speaker
        v
      Storage
```

If you type a letter in a word processor:

1. The keyboard sends input.
2. The CPU processes the key press.
3. RAM temporarily stores the open document.
4. The monitor shows the typed letter.
5. The SSD or hard disk stores the saved file.

#### Hardware

**Hardware** means the physical parts of a computer that can be seen and touched.

| Component | Meaning | Function | Examples |
|---|---|---|---|
| CPU | Central Processing Unit | Processes instructions | Intel Core, AMD Ryzen |
| RAM | Random Access Memory | Temporary working memory | 4 GB, 8 GB, 16 GB |
| ROM | Read Only Memory | Stores boot instructions | BIOS/UEFI chip |
| Motherboard | Main circuit board | Connects all components | Desktop motherboard |
| Storage | Permanent data storage | Saves files and programs | HDD, SSD, pen drive |
| Input Devices | Send data to computer | Enter text, image, sound | Keyboard, mouse, scanner |
| Output Devices | Show result from computer | Display, print, audio | Monitor, printer, speaker |
| VDU | Visual Display Unit | Screen output | LCD, LED monitor |

#### Software

**Software** is a set of programs and instructions that tell hardware what to do. Software cannot be touched physically, but it controls the computer.

| Type | Definition | Purpose | Examples |
|---|---|---|---|
| System Software | Software that controls hardware and runs the computer | Manages resources | Windows, Linux, Android |
| Application Software | Software used for user tasks | Documents, browsing, design | Word processor, browser, spreadsheet |
| Utility Software | Software used for maintenance and protection | Clean, backup, protect | Antivirus, disk cleanup |
| Firmware | Software stored in hardware | Start and control device | BIOS/UEFI, router firmware |

#### Hardware vs Software

| Basis | Hardware | Software |
|---|---|---|
| Definition | Physical parts of a computer | Programs and instructions |
| Can be touched? | Yes | No |
| Purpose | Provides physical working parts | Controls and uses hardware |
| Damage | Can break physically | Can become corrupted or infected |
| Examples | Keyboard, CPU, monitor, printer | Windows, LibreOffice, Chrome |

#### System Software vs Application Software

| Basis | System Software | Application Software |
|---|---|---|
| Meaning | Runs and manages the computer | Helps users complete specific tasks |
| Runs When | Usually starts with computer | Starts when user opens it |
| Dependence | Needed for application software | Depends on system software |
| Examples | Windows, Linux, device drivers | Word, Excel, browser, VLC |

#### Common Mistakes

- Software is not the same as a document file. A document is created using software.
- A monitor is output hardware; it does not process data.
- The cabinet is not the CPU. The CPU is a chip inside the cabinet.

#### Did You Know?

Firmware is why a computer can begin the startup process before Windows or Linux fully loads.

### 2.6 Central Processing Unit (CPU)

The **Central Processing Unit (CPU)** is the main processing component of a computer. It reads instructions, performs calculations, makes decisions, and coordinates other parts of the system. It is often called the brain of the computer because almost every action requires CPU involvement.

#### Main Parts of CPU

| Part | Full Form | Function |
|---|---|---|
| ALU | Arithmetic Logic Unit | Performs arithmetic and logical operations |
| CU | Control Unit | Directs and controls operations |
| Registers | Small high-speed memory | Stores data temporarily inside CPU |

#### How CPU Works

When you open a file, the operating system sends instructions to the CPU. The CPU fetches the instructions, decodes what needs to be done, executes the operation, and sends results to memory, storage, or output devices.

#### CPU Speed and Performance

CPU speed is often measured in **GHz (gigahertz)**. A higher GHz may mean faster processing, but performance also depends on:

- Number of cores
- Processor generation
- Cache memory
- RAM capacity
- Storage speed
- Operating system and software efficiency

#### Real-Life Use

- Opening apps and websites
- Running spreadsheets
- Playing educational videos
- Editing photos or videos
- Processing bank transactions
- Running billing software in shops

#### Exam Tip

CPU has three important parts: **ALU, CU, and Registers**. Remember: **A Computer Runs** = ALU, Control Unit, Registers.

#### Did You Know?

Modern CPUs can contain billions of tiny transistors on one chip.

### 2.7 Memory

**Memory** is used to store data and instructions. Some memory is temporary, and some is permanent.

#### Types of Memory

| Memory | Full Form | Volatile? | Purpose | Example |
|---|---|---|---|---|
| RAM | Random Access Memory | Yes | Temporary working area | Open apps and files |
| ROM | Read Only Memory | No | Stores startup instructions | BIOS/UEFI |
| Cache | - | Usually yes | Very fast memory near CPU | CPU cache |
| Virtual Memory | - | Uses storage | Extends RAM when RAM is low | Page file |

#### RAM vs ROM

| Basis | RAM | ROM |
|---|---|---|
| Full Form | Random Access Memory | Read Only Memory |
| Nature | Temporary | Permanent |
| Volatile? | Yes, data clears when power is off | No, data remains |
| Main Use | Runs current programs | Stores boot instructions |
| Speed | Fast | Slower than RAM for normal operations |
| Example | 8 GB RAM in laptop | BIOS/UEFI chip |

#### Memory Units

| Unit | Value |
|---|---|
| 1 Bit | Smallest unit, 0 or 1 |
| 1 Byte | 8 bits |
| 1 KB | 1024 bytes |
| 1 MB | 1024 KB |
| 1 GB | 1024 MB |
| 1 TB | 1024 GB |

#### Why Memory Matters

If RAM is low, the computer may become slow when many programs are open. If storage is full, you may not be able to save new files. If cache is efficient, the CPU can access frequent data faster.

#### Common Mistakes

- RAM is not permanent storage.
- ROM is not used for storing personal documents.
- A 1 TB hard disk does not automatically make a slow CPU fast.

#### Did You Know?

When you restart a computer, RAM is cleared, but saved files remain on storage.

**Storage** is used to save data, files, programs, and operating systems permanently or for long-term use.

#### Common Storage Devices

| Device | Technology | Speed | Portability | Common Use |
|---|---|---|---|---|
| HDD | Magnetic disk | Slower | Internal/external | Large low-cost storage |

| Basis | HDD | SSD |
|---|---|---|

- Scan pen drives before opening files.
- Do not keep the only copy of important documents on one phone.

### 2.10 Data, Information, Files, and Folders

**Data** means raw facts without full meaning. **Information** means processed data that is useful and meaningful.

Example:

- Data: `85, 90, 78, 92`
- Information: "The student's average mark is 86.25."

#### Data vs Information

| Basis | Data | Information |
|---|---|---|
| Meaning | Raw facts | Processed meaningful result |
| Form | Numbers, text, images, symbols | Report, summary, conclusion |
| Usefulness | Limited until processed | Useful for decision-making |
| Example | List of marks | Average mark and rank |

#### Files and Folders

A **file** is a named collection of data stored on a storage device. A **folder** or **directory** is a container used to organize files and subfolders.

Common file extensions:

| Extension | File Type |
|---|---|
| `.txt` | Text file |
| `.docx`, `.odt` | Document |
| `.xlsx`, `.ods` | Spreadsheet |
| `.pptx`, `.odp` | Presentation |
| `.pdf` | Portable document |
| `.jpg`, `.png` | Image |
| `.mp4` | Video |

#### File Management Tips

- Use meaningful file names, such as `Module-1-Notes.docx`.
- Create subject-wise folders.
- Avoid deleting unknown system files.
- Keep backups of certificates, mark sheets, and ID proof.

#### Did You Know?

File extensions help the operating system decide which application should open a file.

### 2.11 Number Systems

A **number system** is a method of representing numbers using digits and a base. Computers internally use binary because electronic circuits can easily represent two states: ON and OFF.

| Number System | Base | Digits | Use |
|---|---|---|---|
| Decimal | 10 | 0-9 | Daily counting and money |
| Binary | 2 | 0, 1 | Computer internal processing |
| Octal | 8 | 0-7 | Some computing contexts |
| Hexadecimal | 16 | 0-9, A-F | Memory addresses, color codes |

#### Binary Example

`1010` in binary means:

```text
1 x 8 + 0 x 4 + 1 x 2 + 0 x 1 = 10
```

So, binary `1010` = decimal `10`.

#### Why Binary Is Important

Computers use electrical signals. A signal can be represented as ON or OFF, true or false, 1 or 0. Therefore, binary is suitable for computer circuits.

#### Exam Tip

Remember: **Bit** is the smallest unit and stores either `0` or `1`.

#### Did You Know?

Hexadecimal is often used in web color codes. For example, `#FFFFFF` represents white.

### 2.12 Connecting, Starting, and Shutting Down a Computer

Correct handling protects hardware, data, and user safety.

#### Connecting a Desktop PC

1. Place CPU cabinet, monitor, keyboard, and mouse on a stable table.
2. Connect monitor cable such as HDMI or VGA to the CPU cabinet.
3. Connect keyboard and mouse to USB ports.
4. Connect speaker, printer, or scanner if needed.
5. Connect power cables to UPS or power socket.
6. Switch on UPS or stabilizer first if available.
7. Press the power button of CPU and monitor.
8. Wait for the operating system login screen.

#### What Happens During Startup?

1. Power reaches the motherboard.
2. BIOS/UEFI firmware starts.
3. Basic hardware checking happens.
4. The operating system loads from storage into RAM.
5. The login screen or desktop appears.

#### Safe Shutdown

1. Save all work.
2. Close open applications.
3. Click Start -> Power -> Shut down.
4. Wait until screen and CPU turn off.
5. Switch off UPS after shutdown completes.

#### Common Mistakes

- Do not unplug the computer while it is running.
- Do not switch off power without saving work.
- Do not touch cables with wet hands.
- Do not block ventilation holes.

#### Digital Safety Guidance

- Use strong passwords and do not share them.
- Do not open unknown email attachments.
- Use antivirus or built-in security tools.
- Keep important data backed up.
- Log out after using public computers.
- Avoid downloading pirated software.

#### Did You Know?

Improper shutdown may corrupt open files or cause operating system errors.

## 9. Chapter Summary

### Key Takeaways

- A computer accepts input, processes it, stores data, and produces output.
- Hardware means physical parts; software means programs and instructions.
- CPU performs processing and contains ALU, CU, and registers.
- RAM is temporary working memory, while ROM stores startup instructions.
- Input devices send data to the computer; output devices present results.
- Storage devices save files and programs permanently.
- Data becomes information after processing.
- Computers use binary internally because electronic circuits work with two states.
- Safe shutdown, strong passwords, malware awareness, and backup are important for digital safety.

### Important Definitions

| Term | Definition |
|---|---|
| Computer | Electronic device that accepts input, processes data, stores data, and gives output |
| Hardware | Physical components of a computer |
| Software | Programs and instructions used by a computer |
| CPU | Main processing unit of a computer |
| RAM | Temporary volatile memory used while working |
| ROM | Non-volatile memory containing startup instructions |
| Firmware | Software stored in hardware |
| Data | Raw facts |
| Information | Processed meaningful data |
| File | Named collection of data |
| Folder | Container for files and subfolders |
| IoT | Internet-connected devices that collect or exchange data |
| Backup | Extra copy of important data |

### Formulae and Conversions

- 1 Byte = 8 bits
- 1 KB = 1024 bytes
- 1 MB = 1024 KB
- 1 GB = 1024 MB
- 1 TB = 1024 GB
- Binary digits: 0 and 1
- Decimal digits: 0 to 9

### Exam Revision Notes

- Practice RAM vs ROM, HDD vs SSD, hardware vs software, and data vs information.
- Memorize examples of input and output devices.
- Understand startup and shutdown steps.
- Practice simple binary-to-decimal conversions.
