## 1. Introduction to Word Processing

### Explain What It Is

**Word processing** means creating, editing, formatting, storing, printing, and sharing text-based documents using software. Microsoft Word is a word processing application used to prepare letters, resumes, reports, notices, certificates, invoices, assignments, project files, and official documents.

A word processor is different from a plain text editor such as Notepad. Notepad mainly stores simple text, while MS Word supports formatting, tables, images, page layout, headers, footers, spell check, templates, styles, references, and PDF export.

### Explain Why It Is Used

MS Word is used because documents in schools, colleges, offices, banks, hospitals, government departments, and businesses must be clear, professional, printable, and easy to edit. Word helps users prepare documents faster and with better formatting.

### Demonstrate in MS Word

1. Click Start.
2. Search **Word**.
3. Open **Microsoft Word**.
4. Click **Blank document**.
5. Type a heading: `Uses of Computer`.
6. Press Ctrl + S.
7. Choose a folder and save as `uses-of-computer.docx`.

### Writer Equivalent

LibreOffice Writer is an open-source alternative. It can create similar documents and usually saves in `.odt` format, though it can also open and save `.docx` with possible layout differences.

### Real-World Usage

- School: assignments, notices, question papers.
- College: project reports, seminar documents.
- Government office: applications, memos, letters.
- Bank: customer forms and internal reports.
- Hospital: medical certificates and patient instructions.
- Corporate office: proposals, meeting notes, policy documents.
- Freelancing: resume writing, content writing, typing work.

### Practice Exercise

Create a blank Word document titled `My First MS Word Document`. Type five lines about yourself, save the file, close Word, reopen the file, and add one more paragraph.

## 2. MS Word Interface and Screen Layout

### Explain What It Is

The MS Word interface is the working environment where users type, edit, format, and manage documents. It contains the ribbon, tabs, groups, document area, ruler, status bar, and zoom controls.

### MS Word Screen Areas

| Area | Purpose |
|---|---|
| Title Bar | Shows document name and window controls |
| Quick Access Toolbar | Provides quick Save, Undo, Redo commands |
| Ribbon | Main command area with tabs such as Home, Insert, Layout, Review |
| Tabs | Organize commands into categories |
| Groups | Smaller command groups such as Font, Paragraph, Styles |
| Ruler | Helps set margins, tabs, and indents |
| Document Area | Main typing and editing space |
| Status Bar | Shows page number, word count, language, and view options |
| Zoom Slider | Increases or decreases display size |

### Demonstrate in MS Word

1. Open Word.
2. Observe the **Home** tab.
3. Identify the **Font**, **Paragraph**, and **Styles** groups.
4. Turn ruler on: View -> Ruler.
5. Check word count on the status bar.
6. Change zoom using the slider at the bottom-right.

### Writer Equivalent

Writer uses menus, toolbars, and a sidebar instead of Word's ribbon-centered layout. The features are similar, but command locations may differ.

### Common Mistakes

- Typing without saving the document.
- Not noticing the file name in the title bar.
- Confusing zoom level with actual font size.
- Using spaces repeatedly instead of alignment or indents.

## 3. Document Management in MS Word

### Explain What It Is

Document management includes creating, opening, saving, saving as, printing, sharing, and exporting documents. These are basic operations every Word user must master.

### Why It Is Used

Proper document management prevents data loss, maintains file organization, and ensures documents can be shared professionally.

### MS Word Method

#### New Document

1. File -> New.
2. Choose **Blank document** or a template.
3. Shortcut: Ctrl + N.

#### Open Document

1. File -> Open.
2. Browse the saved location.
3. Select the file.
4. Shortcut: Ctrl + O.

#### Save Document

1. File -> Save or press Ctrl + S.
2. Choose folder.
3. Enter file name.
4. Choose file type `.docx`.

#### Save As

Use **Save As** when you want a new copy, a new file name, or a different format such as PDF.

Steps:

1. File -> Save As.
2. Choose location.
3. Enter new file name.
4. Choose file type.
5. Click Save.

#### Print

1. File -> Print or Ctrl + P.
2. Select printer.
3. Choose pages and copies.
4. Check preview.
5. Click Print.

#### Export PDF

1. File -> Save As.
2. Choose file type: PDF.
3. Click Save.

### Writer Equivalent

Writer uses File -> New, Open, Save, Save As, Print, and Export As -> Export as PDF.

### Real-World Usage

- Saving a resume as `.docx` for editing and PDF for submission.
- Printing an application letter.
- Saving project reports in subject-wise folders.
- Creating separate versions such as `Resume_Draft.docx` and `Resume_Final.pdf`.

### Best Practices

- Save every 5-10 minutes.
- Use meaningful names: `Leave_Application_Rasmita.docx`.
- Keep editable `.docx` and final `.pdf` copies.
- Avoid names like `new final latest final.docx`.

### Practice Exercise

Create a document called `document-management-practice.docx`. Save it, use Save As to make a second copy, export it as PDF, and print preview it.

## 4. Basic Text Entry and Editing

### Explain What It Is

Text entry means typing content into a Word document. Editing means correcting, deleting, moving, copying, and improving that content.

### Why It Is Used

Every office document requires accurate typing and editing. Mistakes in official letters, resumes, or applications can create a poor impression.

### MS Word Method

| Task | Method | Shortcut |
|---|---|---|
| New paragraph | Press Enter | Enter |
| Line break in same paragraph | Press Shift + Enter | Shift + Enter |
| Delete left character | Backspace | Backspace |
| Delete right character | Delete | Delete |
| Select all | Home -> Select -> Select All | Ctrl + A |
| Copy | Home -> Copy | Ctrl + C |
| Cut | Home -> Cut | Ctrl + X |
| Paste | Home -> Paste | Ctrl + V |
| Undo | Quick Access Toolbar -> Undo | Ctrl + Z |
| Redo | Quick Access Toolbar -> Redo | Ctrl + Y |

### Real-World Usage

- Correcting a spelling mistake in a leave application.
- Moving a paragraph in a report.
- Copying address details into multiple letters.
- Undoing accidental deletion.

### Common Mistakes

- Pressing Enter many times to create space instead of using paragraph spacing.
- Using Spacebar repeatedly to align text.
- Forgetting to select text before applying formatting.
- Deleting content without using Undo.

### Practice Exercise

Type a paragraph on `Importance of Computer Education`. Copy the first sentence, paste it at the end, undo the paste, redo it, and then delete it.

## 6. MS Word Feature Deep-Dive

### 6.1 Fonts

#### Explain What It Is

A **font** is the design style of text. Examples include Calibri, Arial, Times New Roman, Cambria, and Verdana.

#### Purpose

Fonts improve readability, professionalism, and visual identity.

#### MS Word Steps

1. Select text.
2. Home -> Font group.
3. Choose font name.
4. Choose size.

#### Use Cases

- Resume: Calibri, Arial, or Aptos.
- Formal letter: Times New Roman or Arial.
- Notice: clear bold heading with readable body.

#### Common Mistakes

- Using too many fonts in one document.
- Using decorative fonts in official documents.
- Using very small or very large body text.

#### Best Practices

- Use 1-2 fonts only.
- Body text: 11 or 12 pt.
- Headings: 14-18 pt.
- Keep official documents simple.

## 10. Document Design Best Practices

### Professional Formatting Rules

- Keep formatting consistent.
- Use proper margins.
- Avoid unnecessary colors.
- Use headings and subheadings.
- Use tables for structured data.
- Proofread before printing or sending.

### Font Selection

- Formal documents: Times New Roman, Arial, Calibri, Aptos.
- Body size: 11 or 12 pt.
- Heading size: 14-18 pt.
- Avoid decorative fonts in resumes or letters.

### Heading Hierarchy

| Level | Example |
|---|---|
| Title | Project Report on Computer Education |
| Heading 1 | Introduction |
| Heading 2 | Objectives |
| Heading 3 | Methodology |

### White Space Usage

White space means blank space around text. It improves readability. Do not fill every part of the page with text.

### Alignment Rules

- Titles: center.
- Body: left or justified.
- Dates: left or right, depending on format.
- Tables: text left, numbers center/right.

### Print-Friendly Design

- Use black or dark text.
- Avoid low-contrast colors.
- Check print preview.
- Use A4 page size.
- Keep margins clear.

### PDF Export Standards

- Export final documents as PDF.
- Check file name before sharing.
- Open PDF once to verify layout.
- Use professional names such as `Rasmita_Sahu_Resume.pdf`.

## 11. Mail Merge in MS Word

### What Is Mail Merge?

Mail Merge is an MS Word feature used to create many personalized documents from one common template and a list of records.

### Why Organizations Use It

Organizations use mail merge to save time and reduce typing mistakes when sending similar documents to many people.

### Components

| Component | Meaning |
|---|---|
| Main Document | The common letter/template |
| Data Source | List of names, addresses, marks, salary, etc. |
| Merge Fields | Placeholders such as Name, Address, Roll No |

### Complete MS Word Mail Merge Walkthrough

#### Example: Invitation Letter

1. Create an Excel list with columns: `Name`, `Address`, `Event`, `Date`.
2. Open MS Word.
3. Type the common invitation letter.
4. Go to Mailings -> Start Mail Merge -> Letters.
5. Click Select Recipients -> Use an Existing List.
6. Select the Excel file.
7. Place cursor where name should appear.
8. Click Insert Merge Field -> Name.
9. Insert Address, Event, and Date fields.
10. Click Preview Results.
11. Click Finish & Merge -> Edit Individual Documents or Print Documents.

### Real Examples

- Admit cards.
- Salary slips.
- Certificates.
- Bulk letters.
- Invitations.
- Fee reminders.
- Appointment letters.

### Common Mistakes

- Column names in Excel are unclear.
- Merge fields inserted in wrong position.
- Data source file moved after linking.
- Not previewing before printing.
