   c) Unified Resource Line
   d) User Resource Locator
   **Answer: a) Uniform Resource Locator**
   **Explanation:** A URL (Uniform Resource Locator) is the web address you type in the browser — e.g. https://www.google.com.

97. ISP provides?
   a) Internet connection
   b) Word license
   c) Printer
   d) CPU
   **Answer: a) Internet connection**
   **Explanation:** Internet connection is the correct answer. Word license and Printer are not the right choices here. This is an important CCC exam point — memorise it.

98. HTTPS indicates?
   a) Secure connection
   b) Slow site
   c) No images
   d) Free Wi-Fi
   **Answer: a) Secure connection**
   **Explanation:** HTTPS encrypts data between your browser and the website — look for the padlock when entering passwords or paying online.

99. HTTP stands for?
   a) Hyper Text Transfer Protocol
   b) High Transfer Text Program
   c) Hyper Tool Transfer
   d) Home Text Protocol
   **Answer: a) Hyper Text Transfer Protocol**
   **Explanation:** HTTP/HTTPS are protocols browsers use to load web pages — HTTPS adds encryption for security.

100. Download means?
   a) Save from internet to PC
   b) Upload
   c) Delete
   d) Print
   **Answer: a) Save from internet to PC**
   **Explanation:** Downloading copies a file from the Internet to your computer (e.g. saving a PDF or photo).
