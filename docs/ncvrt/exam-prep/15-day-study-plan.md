# 15-Day CCC Study Plan — Pass with Good Marks

**For:** NITA Classes DCA / CCC students  
**Daily study:** 4–6 hours (theory + practical)  
**Goal:** Complete syllabus + 3 mock tests before NIELIT CCC exam

---

## Days 1–2: Computer Fundamentals (Module 1)

**Study:** Evolution of computers, hardware vs software, CPU, RAM, ROM, storage, input/output, mobile apps, IT gadgets  
**Practical:** Label PC parts; convert decimal ↔ binary; create folder with student details  
**Practice:** Module 1 MCQs (50) — check answers after completing  

**Important questions:**
- Brain of computer? → **CPU**
- Temporary memory? → **RAM**
- Device to enter data? → **Keyboard** (input device)

---

## Days 3–4: Operating System (Module 2)

**Study:** Windows & Linux basics, desktop, taskbar, shortcuts, files/folders, install/remove programs  
**Practical:** Create/rename/copy/move/delete folders; zip folder; Win+L lock; Task Manager  
**Practice:** Module 2 MCQs  

**Practice tasks:** Create Folder · Rename · Copy/Move/Delete files

---

## Days 5–6: Word Processing (Module 3)

**Study:** MS Word / Writer — formatting, tables, images, mail merge, PDF export  
**Practical:** Create **resume** + **formal letter** + **4×3 table**  
**Practice:** Module 3 MCQs  

**Shortcuts:** Ctrl+S Save · Ctrl+B Bold · Ctrl+Enter Page break

---

## Days 7–8: Spreadsheet (Module 4)

**Study:** Excel/Calc — formulas, SUM, AVERAGE, MAX, MIN, COUNT, IF, charts, sort, filter  
**Practical:** Marks sheet 8 students × 3 subjects with Total, Average, Pass/Fail  
**Practice:** Module 4 MCQs  

**Functions:** `=SUM()` · `=AVERAGE()` · `=MAX()` · `=MIN()` · `=COUNT()`

---

## Days 9–10: Presentation (Module 5)

**Study:** PowerPoint/Impress — slides, themes, animation, transition, slideshow  
**Practical:** **5-slide presentation:** About Yourself · Education · Skills · Career Goal · Thank You  
**Practice:** Module 5 MCQs  

**Run slideshow:** F5

---

## Days 11–12: Internet & Communication (Modules 6–7)

**Study:** Browsers, search, downloads, HTTPS; email, attachments, cloud, video meetings  
**Practical:** Bookmark sites; send email with attachment; upload file to Google Drive  
**Practice:** Module 6 + 7 MCQs  

**Terms:** WWW · URL · HTTP/HTTPS · CC · BCC

---

## Day 13: Digital Financial Services (Module 8)

**Study:** UPI, BHIM, NPCI, net banking, wallets, AePS, QR payments, OTP safety  
**Practical:** Demo UPI app screens (no real PIN entry in class); identify phishing SMS  
**Practice:** Module 8 MCQs  

**Important:** UPI = **Unified Payments Interface**

---

## Day 14: E-Governance & Cyber Security (Modules 9–10)

**Study:** DigiLocker, UMANG, .gov.in portals; virus, phishing, strong passwords, 2FA  
**Practical:** Explore India.gov.in; create strong password; list 5 phishing red flags  
**Practice:** Module 9 + 10 MCQs  

**Strong password example style:** `Nita@2026#AI`

---

## Day 15: Future Skills + Mock Tests (Modules 11 + Exam Prep)

**Study:** AI basics, IoT, cloud, digital literacy (Module 11)  
**Morning:** Revise **50 Most Important MCQs**  
**Afternoon:** **Mock Test 1** (100 MCQ, 90 min timed)  
**Evening:** Check Mock Test 1 answer key; revise wrong topics  

---
