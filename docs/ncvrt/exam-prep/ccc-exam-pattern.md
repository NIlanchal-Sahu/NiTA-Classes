# NIELIT CCC Exam Pattern (Revision 4)

**Applicable from:** October 2023 onward  
**Course:** Course on Computer Concepts (CCC)  
**Aligned DCA course at NITA:** 90 Hours (30 Theory + 60 Practical)

---

## Examination Format

| Item | Detail |
|------|--------|
| Question type | Multiple Choice Questions (MCQ) only |
| Total questions | **100** |
| Duration | **90 minutes** |
| Negative marking | **No** |
| Passing marks | **50%** (minimum 50 correct) |
| Mode | Online at NIELIT-approved exam centres |

---

## Syllabus Weightage (11 Modules)

1. Introduction to Computer  
2. Operating System  
3. Word Processing (MS Word / LibreOffice Writer)  
4. Spreadsheet (MS Excel / LibreOffice Calc)  
5. Presentation (PowerPoint / Impress)  
6. Internet & Web Browsing  
7. Communication & Collaboration  
8. Digital Financial Services  
9. E-Governance Services  
10. Cyber Security  
11. Future Skills  

Questions are distributed across all modules. Practical skills (Word, Excel, PPT) are tested conceptually via MCQs.

---

## Passing Criteria

- Score **50 or more** out of 100  
- No section-wise minimum in standard CCC pattern (confirm with latest NIELIT notification)  
- Certificate issued after successful completion of course hours + exam

---

## Exam Day Tips

1. Read all four options before selecting  
2. Attempt easy questions first; mark doubtful for review  
3. Manage time: ~54 seconds per question average  
4. Do not leave questions blank (no negative marking)  
5. Revise **50 Most Important MCQs** and **3 mock tests** before exam
